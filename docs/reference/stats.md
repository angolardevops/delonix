# `delonix stats`

> Live dashboard of resource usage (CPU/MEM/NET/IO/PIDS)

```text
Live dashboard of resource usage (CPU/MEM/NET/IO/PIDS)

Usage: delonix stats [OPTIONS]

Options:
      --no-stream            Print a single snapshot instead of updating live
      --interval <INTERVAL>  Refresh interval (ms) [default: 1000]
      --pt-pt                Show help in Portuguese (pt-PT) instead of English
  -h, --help                 Print help
```
