# `delonix exec`

> Run a command inside a running container

```text
Run a command inside a running container

Usage: delonix exec [OPTIONS] <CONTAINER> <COMMAND>...

Arguments:
  <CONTAINER>   
  <COMMAND>...  

Options:
  -i, --interactive  Keep STDIN open (interactive)
  -t, --tty          Allocate a pseudo-TTY (for interactive shells)
      --pt-pt        Show help in Portuguese (pt-PT) instead of English
  -h, --help         Print help
```
