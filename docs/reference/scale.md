# `delonix scale`

> Set the replica count of a stack service

```text
Set the replica count of a stack service

Usage: delonix scale [OPTIONS] <FILE> <SERVICE> <REPLICAS>

Arguments:
  <FILE>      Path to the stack YAML
  <SERVICE>   Service name
  <REPLICAS>  Desired replica count

Options:
      --pt-pt  Show help in Portuguese (pt-PT) instead of English
  -h, --help   Print help
```
