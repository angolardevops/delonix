# `delonix login`

> Log in to an OCI registry (stores credentials for private images)

```text
Log in to an OCI registry (stores credentials for private images)

Usage: delonix login [OPTIONS] [REGISTRY]

Arguments:
  [REGISTRY]  Registry (default: `docker.io`)

Options:
  -u, --username <USERNAME>  Username
  -p, --password <PASSWORD>  Password (or use `--password-stdin`)
      --password-stdin       Read the password from stdin (for scripts/CI)
      --pt-pt                Show help in Portuguese (pt-PT) instead of English
  -h, --help                 Print help
```
