# `delonix cri`

> Start the CRI server (gRPC) so a kubelet/crictl uses Delonix as runtime

```text
Start the CRI server (gRPC) so a kubelet/crictl uses Delonix as runtime

Usage: delonix cri [OPTIONS]

Options:
      --addr <ADDR>  Unix socket to listen on [default: /run/delonix-cri.sock]
      --pt-pt        Show help in Portuguese (pt-PT) instead of English
  -h, --help         Print help
```
