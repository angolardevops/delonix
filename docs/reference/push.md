# `delonix push`

> Push a local image to an OCI registry (needs `delonix login <host>`)

```text
Push a local image to an OCI registry (needs `delonix login <host>`)

Usage: delonix push [OPTIONS] <IMAGE> [DESTINATION]

Arguments:
  <IMAGE>        Local image to push (name, tag or id prefix)
  [DESTINATION]  Destination reference (default: the image itself, e.g. `me/app:1`)

Options:
      --pt-pt  Show help in Portuguese (pt-PT) instead of English
  -h, --help   Print help
```
