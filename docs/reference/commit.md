# `delonix commit`

> Commit a container's filesystem changes as a new image

```text
Commit a container's filesystem changes as a new image

Usage: delonix commit [OPTIONS] <CONTAINER> <TAG>

Arguments:
  <CONTAINER>  
  <TAG>        New image tag (e.g. `myapp:committed`)

Options:
      --pt-pt  Show help in Portuguese (pt-PT) instead of English
  -h, --help   Print help
```
