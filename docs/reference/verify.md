# `delonix verify`

> Verify an image's cosign/sigstore signature against a public key

```text
Verify an image's cosign/sigstore signature against a public key

Usage: delonix verify [OPTIONS] --key <KEY> <IMAGE>

Arguments:
  <IMAGE>  Image reference (`registry/repo:tag`)

Options:
      --key <KEY>  Cosign public key (PEM)
      --pt-pt      Show help in Portuguese (pt-PT) instead of English
  -h, --help       Print help
```
