# `delonix oci-export`

> Export an image as an OCI runtime bundle (config.json + rootfs) for runc

```text
Export an image as an OCI runtime bundle (config.json + rootfs) for runc

Usage: delonix oci-export [OPTIONS] <IMAGE> <DIR>

Arguments:
  <IMAGE>  
  <DIR>    Output bundle directory

Options:
      --pt-pt  Show help in Portuguese (pt-PT) instead of English
  -h, --help   Print help
```
