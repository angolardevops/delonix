# `delonix ssh`

> Open an interactive shell inside a running container (like `ssh`)

```text
Open an interactive shell inside a running container (like `ssh`)

Usage: delonix ssh [OPTIONS] <CONTAINER> [COMMAND]...

Arguments:
  <CONTAINER>   
  [COMMAND]...  Command to run instead of a login shell (optional)

Options:
      --pt-pt  Show help in Portuguese (pt-PT) instead of English
  -h, --help   Print help
```
