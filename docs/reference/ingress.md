# `delonix ingress`

> Start an L7 ingress (reverse proxy by Host) for a stack

```text
Start an L7 ingress (reverse proxy by Host) for a stack

Usage: delonix ingress [OPTIONS] <FILE>

Arguments:
  <FILE>  Path to the stack YAML

Options:
      --addr <ADDR>  Listen address [default: 127.0.0.1:8080]
      --pt-pt        Show help in Portuguese (pt-PT) instead of English
  -h, --help         Print help
```
