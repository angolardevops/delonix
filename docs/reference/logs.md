# `delonix logs`

> Show a container's captured stdout/stderr (detached)

```text
Show a container's captured stdout/stderr (detached)

Usage: delonix logs [OPTIONS] <CONTAINER>

Arguments:
  <CONTAINER>  

Options:
  -f, --follow       Follow log output (stream new lines, like `tail -f`)
      --tail <TAIL>  Show only the last N lines
      --pt-pt        Show help in Portuguese (pt-PT) instead of English
  -h, --help         Print help
```
