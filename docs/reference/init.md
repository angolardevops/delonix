# `delonix init`

> Scaffold a complete project (`Delonixfile` + `delonix-stack.yaml`)

```text
Scaffold a complete project (`Delonixfile` + `delonix-stack.yaml`)

Usage: delonix init [OPTIONS] [IMAGE]

Arguments:
  [IMAGE]  Base image for the `FROM` (default: alpine:latest)

Options:
      --name <NAME>  Project/stack name (default: the directory name)
      --pt-pt        Show help in Portuguese (pt-PT) instead of English
  -h, --help         Print help
```
