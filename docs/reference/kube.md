# `delonix kube`

> Kubernetes interop (generate Pod manifests)

```text
Kubernetes interop (generate Pod manifests)

Usage: delonix kube [OPTIONS] <COMMAND>

Commands:
  generate  Generate a k8s Pod manifest from a container or pod
  help      Print this message or the help of the given subcommand(s)

Options:
      --pt-pt  Show help in Portuguese (pt-PT) instead of English
  -h, --help   Print help
```

## `delonix kube generate`

> Generate a k8s Pod manifest from a container or pod

```text
Generate a k8s Pod manifest from a container or pod

Usage: delonix kube generate [OPTIONS] <NAME>

Arguments:
  <NAME>  Container name/id, or a pod name

Options:
      --pt-pt  Show help in Portuguese (pt-PT) instead of English
  -h, --help   Print help
```
