# `delonix cp`

> Copy files/dirs between host and a container (`SRC DST`; one side `c:/path`)

```text
Copy files/dirs between host and a container (`SRC DST`; one side `c:/path`)

Usage: delonix cp [OPTIONS] <SRC> <DST>

Arguments:
  <SRC>  Source (`path` or `container:/path`)
  <DST>  Destination (`path` or `container:/path`)

Options:
      --pt-pt  Show help in Portuguese (pt-PT) instead of English
  -h, --help   Print help
```
