# `delonix convert`

> Convert a stack to docker-compose or k8s manifests

```text
Convert a stack to docker-compose or k8s manifests

Usage: delonix convert [OPTIONS] --to <TO> <FILE>

Arguments:
  <FILE>  Path to the stack YAML

Options:
      --to <TO>          Target: `docker-compose` or `k8s`
  -o, --output <OUTPUT>  Output file (compose) or directory (k8s); otherwise stdout
      --pt-pt            Show help in Portuguese (pt-PT) instead of English
  -h, --help             Print help
```
